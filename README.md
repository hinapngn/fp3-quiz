# FP3級 学科 問題集

FP技能検定3級 学科試験対策のクイズアプリ(PWA)。aws-saa-quiz と同じ仕組みの、ビルド不要・依存ゼロの静的アプリです。

**2025年度試験対応(法令基準日: 2025年4月1日)**

## 収録内容

全120問(本試験2回分相当)

| 分野 | ○×式 | 三答択一式 |
|---|---|---|
| ライフプランニングと資金計画 | 10問 | 10問 |
| リスク管理 | 10問 | 10問 |
| 金融資産運用 | 10問 | 10問 |
| タックスプランニング | 10問 | 10問 |
| 不動産 | 10問 | 10問 |
| 相続・事業承継 | 10問 | 10問 |

## 機能

- **本試験と同じ問題形式**: ○×式・三答択一式
- **間隔復習**: 間違えた問題は30問後に自動で再出題。正解するとマスター済みに
- **分野別学習モード**: 苦手分野だけを選んで集中練習
- **成績記録**: 分野別正答率・累計成績を localStorage に保存
- **PWA対応**: ホーム画面に追加してオフラインでも利用可能

## ファイル構成

```
index.html    アプリ本体(UI・ロジック)
questions.js  問題データ(ALL_Q 配列)
sw.js         Service Worker(index.html / questions.js はネットワーク優先)
manifest.json PWA マニフェスト
icons/        アプリアイコン
```

## 問題の追加方法

`questions.js` の `ALL_Q` 配列に以下の形式で追加します。既存の記録を持つユーザーにも、新しい問題は自動的に出題キューへ追加されます。

```js
// ○×式
{id:121,theme:"life",diff:1,type:"ox",
 text:"問題文...",choices:OX,correct:["○"],
 explanation:"解説..."},

// 三答択一式
{id:122,theme:"tax",diff:2,type:"three",
 text:"問題文...",
 choices:[{key:"1",text:"..."},{key:"2",text:"..."},{key:"3",text:"..."}],
 correct:["2"],explanation:"解説...",
 wrong_reasons:{"1":"...","3":"..."}},
```

- `theme`: `life` / `risk` / `finance` / `tax` / `estate` / `inherit`
- `diff`: 1(易) / 2(中) / 3(難)
- Service Worker がネットワーク優先で `questions.js` を取得するため、キャッシュ名の更新なしで問題追加が反映されます

## デプロイ

静的ファイルのみなので、GitHub Pages 等でそのまま公開できます(リポジトリの Settings → Pages でブランチを指定)。
